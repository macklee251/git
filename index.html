Curso de Git

Comandos

  • git add *.html coloca no stage totos os arquivos .html
  • git restore remove o arquivo do stage (Volta para a versão do diretório)
  • podemos colocar todos os arquivos no stage e dar commit ao mesmo tempo com o comando git commit -am "Mensagem"
  • podemos ajustar o ultimo commmit que foi feito com o comando git commit --amend -m "Mensagem"
  • podemos ver um gráfico das logs como: git log --graph
  • podemos ver uma versão simplificada dos logs com git shortlog
